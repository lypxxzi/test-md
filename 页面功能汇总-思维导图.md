## 简化版（按模块分组）

### 基础数据与看板
```
物料管理 ────┬─ 列表 ─┬─ 编辑/详情
             │        ├─ 表单A 新增（原料/内包材/外包材）
             │        ├─ 表单B 新增（配方/中间品/产品）
             │        └─ 逻辑删除
             │
BOM管理 ─────┼─ 列表 ─── 新增/编辑（物料明细、工艺路线、工艺参数一页配置）
             │
库存看板 ────└─ 库存及批次综合查询（仓库 - 物料 - 批次 - 可用数量） ⭐新增
```

### 返工单管理
```
返工单列表 ──┬─ 新增 ─┬─ 基本信息（选择不合格成品批次、自动校验库存、选择返工类型）
             │        ├─ BOM配置（关联使用现有BOM或跳转快速生成返工BOM）
             │        └─ 一键下达（分流下达，同步推返工生产单+申购单，可多次下达）
             ├─ 编辑/删除（仅草稿）
             └─ 详情
```

### 申购采购（采购线）
```
申购需求管理 ─┬─ 列表（按生产单合并展示物料需求）
              └─ 详情

申购管理 ────┬─ 列表
             ├─ 申购确认（核心：查看实时多维库存、修改申购数、一键合并转采购）⭐优化
             └─ 详情 / 关闭

采购管理 ────┬─ 列表（认领合并后的采购需求，直接编辑单价、采购数量、预计到货时间）⭐合并简化
             └─ 详情 / 取消采购

回料管理 ────┬─ 新增（仓管：选择采购单，只选入库仓库、不选库位，填实收数）
             └─ 自动生成全新采购批次，库存即刻增加
```

### 生产与发料（执行线）
```
生产任务列表 ─┬─ 查看生产单状态（待领料/生产中/已完成/已关闭）
             ├─ 强制关闭（任意状态一键关闭并终止流转）
             └─ 详情

领退料工作台 ─┬─ 生产员 Tab（发起领料申请-自动带BOM用量、发起退料申请-填退料数和原因）⭐角色分工
             └─ 仓管员 Tab（发料确认-选择发料批次/实发数/扣库存、退料入库确认-增加库存）

生产执行中心 ─┬─ 包装类返工 ──► 一键完工汇报（输入实际完工数即可一键汇报）⭐核心分流
             └─ 配制类返工 ──► 步骤备料投料（工艺步骤备料记录、直接投料/备料后投料）

生产完工入库 ─┬─ 入库发起（生产员：输入入库数，系统自动生成FG+成品批次号）
             └─ 入库确认（仓管员：核对数量，确认入库增加成品库存）
```

## 业务全流程速查

```
计划员创建返工单(校验批次) → 一键下达生产+申购 
                                │
          ┌─────────────────────┴─────────────────────┐
          ▼ 生产单(执行线)                             ▼ 申购单(采购线)
   领料申请(生产员)                               申购确认(申购员比对库存)
          │                                           │
   发料执行(仓管扣减库存)                          合并采购并执行(采购员)
          │                                           │
   生产执行(按返工类型分流 ⭐️)                      回料入库(仓库收料增库存)
   ├── 包装: 输入完工数完工汇报                        │
   └── 配制: 按步骤工艺备料+投料                       │
          │                                           │
   生产入库(自动生成FG批次) ◄─────────────────────────┘(到货原料可用于领料)
          │
   入库确认(仓管增加成品库存)
```
