# 实体设计 · 采购

---

## 1. 申购需求单主表 ⭐生产发起

> 生产单在准备或执行过程中，每一次发起的采购申请单。如：首次BOM自动展开、生产中补料申请、损耗增补等。

| 字段名 | 字段类型 | 是否必填 | 说明 |
|---|---|---|---|
| 需求单号 | varchar | 是 | 主键，业务单号（MR-YYYYMMDD-XXX） |
| 返工生产单号 | varchar | 是 | FK→返工生产单 |
| 需求类型 | tinyint | 是 | 1-初始BOM展开 / 2-生产中补料 / 3-损耗增补 |
| 发起人 | bigint | 是 | FK→用户表 |
| 状态 | tinyint | 是 | 0-待申购/1-申购中/2-部分已申/3-已完成/4-已关闭 |
| 创建时间 | datetime | 是 | |
| 备注 | varchar | 否 | 记录如：为什么需要补料 |

---

## 2. 申购需求明细行 ⭐申请明细

> 每次采购申请中的具体物料及数量

| 字段名 | 字段类型 | 是否必填 | 说明 |
|---|---|---|---|
| 需求明细行ID | bigint | 是 | 主键，自增 |
| 需求单号 | varchar | 是 | FK→申购需求单主表 |
| BOM物料明细ID | bigint | 否 | FK→返工生产BOM物料明细（手工补料时可为空） |
| 物料编码 | varchar | 是 | FK→物料表 |
| 需求数量 | decimal | 是 | 申请要买的数量 |
| 已申购数量 | decimal | 是 | 累积已被申购单引用的数量 |
| 状态 | tinyint | 是 | 0-待申购/1-申购中/2-已申购/3-已关闭 |
| 备注 | varchar | 否 | |

## 3. 申购单主表

> 对应申购需求单 1:1 转换而来的审批单据

| 字段名 | 字段类型 | 是否必填 | 说明 |
|---|---|---|---|
| 申购单号 | varchar | 是 | 主键 |
| 关联需求单号 | varchar | 是 | FK→申购需求单主表 (1:1 关联) |
| 工作流状态 | tinyint | 是 | 0-草稿/1-待确认/2-已确认/3-已关闭 |
| 采购状态 | tinyint | 是 | 0-无需采购/1-未采购/2-部分采购/3-全部采购 |
| 回料状态 | tinyint | 是 | 0-无需回料/1-未回料/2-部分已回/3-全部已回 |
| 申购员 | bigint | 否 | FK→用户表 |
| 创建时间 | datetime | 是 | |
| 关闭时间 | datetime | 否 | |
