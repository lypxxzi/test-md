# 实体设计 · 物料相关

## 1. 物料

| 字段名 | 字段类型 | 是否必填 | 说明 |
|---|---|---|---|
| id | bigint | 是 | 主键 |
| 物料编码 | varchar | 是 | 物料唯一编码 |
| 物料名称 | varchar | 是 | 物料名称 |
