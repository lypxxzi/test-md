# 实体设计 · 入库

---

## 1. 入库主单

| 字段名 | 字段类型 | 是否必填 | 说明 |
|---|---|---|---|
| 入库单号 | varchar | 是 | 主键 |
| 入库类型 | tinyint | 是 | 1-生产入库/2-采购入库/3-调拨入库/4-盘盈 |
| 来源单据类型 | tinyint | 是 | 1-返工生产单/2-回料单/3-手工 |
| 来源单据编号 | varchar | 是 | 回料单号或生产单号 |
| 入库仓库 | varchar | 是 | |
| 状态 | tinyint | 是 | 0-待入库/1-部分入库/2-已入库/3-已关闭 |
| 入库人 | bigint | 是 | FK→用户表 |
| 入库时间 | datetime | 是 | |
| 备注 | varchar | 否 | |

---

## 2. 入库明细表

| 字段名 | 字段类型 | 是否必填 | 说明 |
|---|---|---|---|
| 明细行ID | bigint | 是 | 主键，自增 |
| 入库单号 | varchar | 是 | FK→入库主单 |
| 来源明细行ID | bigint | 否 | 来源单据的具体明细行ID |
| 物料编码 | varchar | 是 | FK→物料表 |
| 物料名称 | varchar | 是 | |
| 规格型号 | varchar | 否 | |
| 批次ID | bigint | 是 | FK→批次表 |
| 库位 | varchar | 否 | FK→库位表 |
| 计划入库数量 | decimal | 否 | 来源明细行带出 |
| 实际入库数量 | decimal | 是 | |
| 单位 | varchar | 是 | |
| 状态 | tinyint | 是 | 0-待入库/1-已入库/2-取消 |
| 实际入库时间 | datetime | 否 | 行级入库完成时间 |
| 关联质检单号 | varchar | 否 | |
| 备注 | varchar | 否 | |

> **入库状态流转**：
> - 主单状态根据明细汇总自动计算：
>   - 全部明细"待入库" → 主单"待入库"
>   - 部分明细"已入库" → 主单"部分入库"
>   - 全部明细"已入库/取消" → 主单"已入库"
> - 支持分批入库：同一入库单可分多次完成不同明细的入库
> - 已部分入库的主单可继续入库剩余明细，或手动关闭
