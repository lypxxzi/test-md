# 实体设计 · 返工单

---

## 1. 返工单主表

| 字段名 | 字段类型 | 是否必填 | 说明 |
|---|---|---|---|
| 返工单号 | varchar | 是 | 主键，业务编号 |
| 计划开始时间 | datetime | 是 | |
| 计划结束时间 | datetime | 是 | |
| 状态 | tinyint | 是 | 0-草稿/1-审批中/2-已审批/3-已驳回 |
| 创建人 | bigint | 是 | FK→用户表 |
| 创建时间 | datetime | 是 | |

---

## 2. 返工单明细行

| 字段名 | 字段类型 | 是否必填 | 说明 |
|---|---|---|---|
| 行号 | int | 是 | 联合主键 |
| 返工单号 | varchar | 是 | FK→返工单主表 |
| 物料编码 | varchar | 是 | FK→物料表 |
| 批次ID | bigint | 是 | FK→批次表 |
| 返工类型 | tinyint | 是 | 1-外包装/2-内包装/3-配制 |
| 返工数量 | decimal | 是 | |
| 已下达数量 | decimal | 是 | 累计下达，不超过返工数量 |
| BOM编号 | varchar | 是 | FK→BOM表 |
| 备注 | varchar | 否 | |

---

## 3. 返工生产单

> 返工单审批通过后，按计划员下达数量生成

| 字段名 | 字段类型 | 是否必填 | 说明 |
|---|---|---|---|
| 返工生产单号 | varchar | 是 | 主键 |
| 返工单号 | varchar | 是 | FK→返工单主表 |
| 返工单明细行号 | int | 是 | FK→返工单明细行 |
| 物料编码 | varchar | 是 | FK→物料表 |
| 批次ID | bigint | 是 | FK→批次表 |
| BOM编号 | varchar | 是 | FK→BOM表 |
| 下达数量 | decimal | 是 | |
| 状态 | tinyint | 是 | 0-草稿/1-待领料/2-生产中/3-已完成/4-已关闭 |
| 强制关闭标记 | tinyint | 否 | 0-否/1-是 |
| 创建时间 | datetime | 是 | |

---

## 4. 返工生产BOM主表 ⭐实际执行BOM

> 返工生产单的实际BOM，记录具体执行时的物料和工艺

| 字段名 | 字段类型 | 是否必填 | 说明 |
|---|---|---|---|
| BOM主表ID | bigint | 是 | 主键，自增 |
| 返工生产单号 | varchar | 是 | FK→返工生产单（1:1） |
| 引用BOM编号 | varchar | 是 | FK→标准BOM表（复制来源） |
| BOM名称 | varchar | 是 | 如"返工生产单RWP001实际BOM" |
| 备注 | varchar | 否 | |

---

## 5. 返工生产BOM物料明细 ⭐子表一

> 实际返工生产用了哪些物料、配比多少

| 字段名 | 字段类型 | 是否必填 | 说明 |
|---|---|---|---|
| 明细ID | bigint | 是 | 主键，自增 |
| BOM主表ID | bigint | 是 | FK→返工生产BOM主表 |
| 物料编码 | varchar | 是 | FK→物料表 |
| 实际配比 | decimal | 是 | 执行时的实际配比% |
| 计划配比 | decimal | 否 | 从标准BOM复制的配比 |
| 物料类型 | tinyint | 是 | 1-原料/2-内包材/3-外包材... |
| 备注 | varchar | 否 | 如有调整注明原因 |

---

## 6. 返工生产工艺步骤 ⭐子表二

> 实际返工生产的工艺步骤

| 字段名 | 字段类型 | 是否必填 | 说明 |
|---|---|---|---|
| 步骤ID | bigint | 是 | 主键，自增 |
| BOM主表ID | bigint | 是 | FK→返工生产BOM主表 |
| 步骤序号 | int | 是 | |
| 步骤描述 | varchar | 是 | 操作说明 |
| 计划步骤ID | bigint | 否 | FK→标准BOM工艺步骤 |
| 备注 | varchar | 否 | |

---

## 7. 返工生产步骤参数 ⭐子表三

> 实际执行时的参数设置

| 字段名 | 字段类型 | 是否必填 | 说明 |
|---|---|---|---|
| 参数ID | bigint | 是 | 主键，自增 |
| 步骤ID | bigint | 是 | FK→返工生产工艺步骤 |
| 参数名称 | varchar | 是 | 如温度/压力/转速 |
| 实际值 | decimal | 否 | 执行时的实际设定值 |
| 计划值 | decimal | 否 | 从标准BOM复制的值 |
| 单位 | varchar | 否 | |
| 备注 | varchar | 否 | |

## 8. 返工生产步骤投料明细 ⭐子表四

> 物料在各步骤的实际投料比
