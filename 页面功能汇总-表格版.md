## 核心流程速查

### 流程1：返工单创建到审批
```
计划员 → 创建返工单 → 选产品/批次/BOM → 下达生产+申购 → 提交审批 → 审批通过
```

### 流程2：领料完整流程
```
生产员 → 领料申请 → 审批 → 仓库发料 → 领料完成
   ↑
   └── 多领时 → 退料申请 → 审批 → 仓库入库
```

### 流程3：申购到回料
```
申购员 → 申购确认(看库存填确认数) → 合并采购 → 采购员采购 → 到货回料入库
```

### 流程4：生产执行（配制返工）
```
步骤1备料 → 步骤1投料 → 步骤2备料 → 步骤2投料 → ... → 生产入库
```
