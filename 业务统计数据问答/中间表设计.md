# 中间表设计

## 表1：`dws_customer_monthly`（客户月度宽表）

**粒度：一行 = 一个租户 + 一个客户 + 一个月**

---

### 维度字段（10列）

| 字段 | 类型 | 说明 |
|------|------|------|
| tenant_id | bigint | 租户ID |
| stat_month | varchar(7) | 统计月份（如 2025-06） |
| customer_id | bigint | 客户ID |
| customer_name | varchar | 客户名称 |
| salesperson_id | bigint | 业务员ID |
| salesperson_name | varchar | 业务员名称 |
| country | varchar | 国家（如：中国、美国、日本） |
| region_type | varchar | 区域类型（domestic=国内 / overseas=国外） |
| province | varchar | 客户所在省份（国内客户） |
| city | varchar | 客户所在城市（国内客户） |

---

### 状态标记字段（4列）

| 字段 | 类型 | 说明 |
|------|------|------|
| is_active | tinyint | 本月是否活跃（1=活跃，0=非活跃） |
| has_order | tinyint | 历史是否下过单（1=下过，0=从未下单） |
| is_reorder | tinyint | 本月是否有返单（1=有，0=无） |
| status_change | varchar | 本月状态变化（no_order_to_active / active_to_inactive / 空） |

---

### 订单量字段（5列）

| 字段 | 类型 | 说明 |
|------|------|------|
| order_count | int | 本月新增订单总数 |
| shipped_order_count | int | 本月已发货订单数 |
| in_production_count | int | 本月在生产中订单数（已有生产指令） |
| unscheduled_count | int | 本月未排产订单数（没有生产指令） |
| reorder_times | int | 本月返单次数 |

---

### 订单额字段（4列）

| 字段 | 类型 | 说明 |
|------|------|------|
| order_amount | decimal(14,2) | 本月新增订单总额 |
| shipped_amount | decimal(14,2) | 本月已发货订单额 |
| in_production_amount | decimal(14,2) | 本月在生产中订单额 |
| unscheduled_amount | decimal(14,2) | 本月未排产订单额 |

---

### 出货字段（2列）

| 字段 | 类型 | 说明 |
|------|------|------|
| shipping_count | int | 本月出货数量 |
| shipping_amount | decimal(14,2) | 本月出货金额 |

---

### 按期交付字段（4列）

| 字段 | 类型 | 说明 |
|------|------|------|
| total_product_count | int | 本月订单产品数（按订单产品统计） |
| completed_count | int | 本月已完成入库订单数 |
| on_time_count | int | 本月按期完成订单产品数 |
| overdue_count | int | 本月逾期未完成订单产品数 |

---

### 预算利润字段（4列）

| 字段 | 类型 | 说明 |
|------|------|------|
| budget_cost | decimal(14,2) | 本月订单预算总成本 |
| budget_profit | decimal(14,2) | 本月订单预算总利润 |
| shipped_budget_cost | decimal(14,2) | 本月已发货订单预算总成本 |
| shipped_budget_profit | decimal(14,2) | 本月已发货订单预算总利润 |

---

### 实际利润字段（2列，已发货订单）

| 字段 | 类型 | 说明 |
|------|------|------|
| actual_cost | decimal(14,2) | 本月已发货订单实际总成本 |
| actual_profit | decimal(14,2) | 本月已发货订单实际总利润 |

---

### 回款字段（2列）

| 字段 | 类型 | 说明 |
|------|------|------|
| collected_amount | decimal(14,2) | 本月已收款金额 |
| pending_amount | decimal(14,2) | 本月待收款金额 |

---

### 应收款字段（5列）

| 字段 | 类型 | 说明 |
|------|------|------|
| receivable_amount | decimal(14,2) | 本月已结算订单总额 |
| remaining_unallocated | decimal(14,2) | 剩余未分配收款总额 |
| prepaid_unallocated | decimal(14,2) | 预收单/全款单未分配总额 |
| monthly_unallocated | decimal(14,2) | 月结单未分配总额 |
| total_uncollected | decimal(14,2) | 合计未收款总额 |

---

### 字段总计：42列

---

## 表2：`dws_customer_product_monthly`（客户-产品月度表）

**粒度：一行 = 一个租户 + 一个客户 + 一个产品 + 一个月**

---

### 维度字段（9列）

| 字段 | 类型 | 说明 |
|------|------|------|
| tenant_id | bigint | 租户ID |
| stat_month | varchar(7) | 统计月份 |
| customer_id | bigint | 客户ID |
| customer_name | varchar | 客户名称 |
| salesperson_id | bigint | 业务员ID |
| salesperson_name | varchar | 业务员名称 |
| product_id | bigint | 产品ID |
| product_name | varchar | 产品名称 |
| product_category | varchar | 产品类型（产品/中间品/配方产品/非化妆品） |

---

### 订单量字段（4列）

| 字段 | 类型 | 说明 |
|------|------|------|
| order_count | int | 该产品本月订单量 |
| shipped_order_count | int | 该产品本月已发货订单数 |
| in_production_count | int | 该产品本月在生产中订单数 |
| unscheduled_count | int | 该产品本月未排产订单数 |

---

### 订单额字段（4列）

| 字段 | 类型 | 说明 |
|------|------|------|
| order_amount | decimal(14,2) | 该产品本月订单金额 |
| shipped_amount | decimal(14,2) | 该产品本月已发货订单额 |
| in_production_amount | decimal(14,2) | 该产品本月在生产中订单额 |
| unscheduled_amount | decimal(14,2) | 该产品本月未排产订单额 |

---

### 按期交付字段（4列）

| 字段 | 类型 | 说明 |
|------|------|------|
| total_product_count | int | 该产品本月订单产品数 |
| completed_count | int | 该产品本月已完成入库数 |
| on_time_count | int | 该产品本月按期完成数 |
| overdue_count | int | 该产品本月逾期数 |

---

### 预算利润字段（4列）

| 字段 | 类型 | 说明 |
|------|------|------|
| budget_cost | decimal(14,2) | 该产品本月预算成本 |
| budget_profit | decimal(14,2) | 该产品本月预算利润 |
| shipped_budget_cost | decimal(14,2) | 该产品本月已发货订单预算成本 |
| shipped_budget_profit | decimal(14,2) | 该产品本月已发货订单预算利润 |

---

### 实际利润字段（2列，已发货订单）

| 字段 | 类型 | 说明 |

---

### 回款字段（2列）

| 字段 | 类型 | 说明 |
|------|------|------|

---

### 应收款字段（5列）

| 字段 | 类型 | 说明 |
|------|------|------|

| remaining_unallocated | decimal(14,2) | 未分配收款总额 |

| monthly_unallocated | decimal(14,2) | 月结单未分配 |

---

## 查询示例

### 从表1查：某客户累计订单额

```sql
SELECT SUM(order_amount) as total_order_amount
FROM dws_customer_monthly
WHERE tenant_id = 'A公司' AND customer_id = 1001
```

### 从表1查：某客户按月订单额趋势

```sql
SELECT stat_month, order_amount
FROM dws_customer_monthly
WHERE tenant_id = 'A公司' AND customer_id = 1001
ORDER BY stat_month
```

### 从表1查：某客户累计回款率

```sql
SELECT SUM(collected_amount) / SUM(order_amount) as collection_rate
FROM dws_customer_monthly
WHERE tenant_id = 'A公司' AND customer_id = 1001
```

### 从表2查：某客户累计订单量TOP30产品

```sql
SELECT product_name, SUM(order_count) as total_orders
FROM dws_customer_product_monthly
WHERE tenant_id = 'A公司' AND customer_id = 1001
GROUP BY product_id, product_name
ORDER BY total_orders DESC
LIMIT 30
```

### 从表2查：某客户某产品按月趋势

```sql
SELECT stat_month, order_count, order_amount
FROM dws_customer_product_monthly
WHERE tenant_id = 'A公司' AND customer_id = 1001 AND product_id = 2001
ORDER BY stat_month
```

### 从表1查：全租户累计订单统计

```sql
SELECT SUM(order_count) as total_orders, SUM(order_amount) as total_amount,
       SUM(shipped_order_count) as shipped_orders, SUM(shipped_amount) as shipped_total
FROM dws_customer_monthly
WHERE tenant_id = 'A公司'
```

### 从表1查：全租户按月订单趋势

```sql
SELECT stat_month, SUM(order_count) as monthly_orders, SUM(order_amount) as monthly_amount
FROM dws_customer_monthly
WHERE tenant_id = 'A公司'
GROUP BY stat_month
ORDER BY stat_month
```

### 从表1查：存量订单地域统计（国内按省份）

```sql
SELECT province, city, SUM(order_count) as orders, SUM(order_amount) as amount
FROM dws_customer_monthly
WHERE tenant_id = 'A公司' AND region_type = 'domestic'
GROUP BY province, city
ORDER BY amount DESC
```

### 从表1查：存量订单地域统计（国外按国别）

```sql
SELECT country, SUM(order_count) as orders, SUM(order_amount) as amount
FROM dws_customer_monthly
WHERE tenant_id = 'A公司' AND region_type = 'overseas'
GROUP BY country
ORDER BY amount DESC
```

### 从表2查：全租户TOP30产品订单量

```sql
SELECT product_name, SUM(order_count) as total_orders
FROM dws_customer_product_monthly
WHERE tenant_id = 'A公司'
GROUP BY product_id, product_name
ORDER BY total_orders DESC
LIMIT 30
```

### 从表2查：按产品类型筛选（只看中间品）

```sql
SELECT SUM(order_count) as total_orders, SUM(order_amount) as total_amount,
       SUM(shipped_order_count) as shipped_orders, SUM(shipped_amount) as shipped_total
FROM dws_customer_product_monthly
WHERE tenant_id = 'A公司' AND customer_id = 1001
  AND product_category = '中间品'
  AND stat_month BETWEEN '2025-01' AND '2025-06'
```

### 从表2查：期初/期末/本周期新增（按产品类型）

```sql
-- 期末累计（截至2025-06）
SELECT SUM(order_count) as period_end_total
FROM dws_customer_product_monthly
WHERE tenant_id = 'A公司' AND customer_id = 1001
  AND product_category = '产品'
  AND stat_month <= '2025-06'

-- 期初累计（截至2024-12）
SELECT SUM(order_count) as period_start_total
FROM dws_customer_product_monthly
WHERE tenant_id = 'A公司' AND customer_id = 1001
  AND product_category = '产品'
  AND stat_month <= '2024-12'

-- 本周期新增（2025-01 ~ 2025-06）
SELECT SUM(order_count) as period_new
FROM dws_customer_product_monthly
WHERE tenant_id = 'A公司' AND customer_id = 1001
  AND product_category = '产品'
  AND stat_month BETWEEN '2025-01' AND '2025-06'
```
