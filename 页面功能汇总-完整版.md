# 页面以及功能汇总（完整版）

> 说明：本版本基于业务流程文档整理，排除库位、供应商、质检、报废相关功能

---

## 一、基础信息

### 1.1 物料管理
- **列表页**
  - 搜索筛选（按物料类型、启用状态）
  - 分页展示
- **新增/编辑页**（按物料类型分页面）
  - 原料、内包材、外包材 —— 一个页面
  - 配方、中间品、产品 —— 一个页面
- **详情页**
  - 基本信息展示
  - 关联BOM列表
- **删除**（逻辑删除，校验无关联后允许）

### 1.2 BOM管理
- **列表页**
  - 搜索筛选（按所属物料、BOM类型、版本）
  - 分页展示
- **新增页**
  - 选择所属物料
  - 填写BOM名称、版本
  - 选择BOM类型（主BOM/返工BOM）
  - 配置物料明细（原料/内包材/外包材/基料/配方产品/中间品/成品）
  - 配置工艺步骤（步骤序号、步骤描述）
  - 配置步骤参数（参数名称、计划值、单位、上下限）
  - 配置步骤投料明细（物料、投料比）
- **编辑页**（同新增，支持修改）
- **详情页**
  - BOM基本信息
  - 物料明细列表
  - 工艺步骤及参数
  - 投料明细

## 二、返工单管理

### 2.1 返工单列表
- **列表页**
  - 搜索筛选（按返工单号、状态、创建时间）
  - 分页展示
  - 操作按钮：编辑/提交审批/详情/删除（仅草稿）
- **状态展示**：草稿/审批中/已审批/已驳回

### 2.2 返工单新增/编辑
- **基本信息**
  - 选择返工产品（物料）
  - 选择产品批次（自动校验库存是否存在）
  - 选择返工类型（外包装返工/内包装返工/配制返工）
  - 填写返工数量
  - 计划开始/结束时间
- **BOM配置**
  - 系统自动带出主BOM
  - 选择使用BOM（从现有BOM选择）
  - 或跳转BOM管理新建返工专用BOM
- **下达生产/申购**
  - 指定本次下达数量
  - 系统自动生成返工生产单（草稿）
  - 系统自动生成申购单（草稿）
  - 可多次下达（累计不超返工数量）

### 2.3 返工单详情
- 基本信息展示
- 明细行列表（产品、批次、返工类型、数量、已下达数量）
- 已下达生产单列表
- 关联申购单列表
- 审批记录

### 2.4 返工单审批
- **审批列表**（审批人视角）
  - 待审批返工单列表
  - 已审批历史
- **审批详情/操作页**
  - 查看返工单完整信息
  - 审批通过：返工生产单和申购单变为可执行状态
  - 审批驳回：返回草稿状态，填写驳回原因

## 三、申购采购

### 3.1 申购需求管理（生产单分组）
- **列表页**
  - 按返工生产单分组展示物料需求
  - 展示字段：生产单号、物料编码、物料名称、需求数量、状态
  - 状态：待申购/已申购/已关闭
- **详情页**
  - 生产单基本信息
  - 物料需求明细列表

### 3.2 申购管理
- **列表页**
  - 搜索筛选（按申购单号、工作流状态、采购状态、回料状态）
  - 分页展示
- **申购确认页**（申购员操作）
  - 查看申购单明细（物料、计划申购数量）
  - 实时查询展示物料多维状态：
    - 仓库库存数量
    - 申购中数量
    - 采购中数量
    - 生产中占用数量
  - 逐行填写【确认申购数量】（可低于计划，可填0）
  - 提交确认
- **处理（合并采）**
  - 多个申购单合并生成采购单
  - 选择需合并的申购明细行
  - 确认合并后生成采购单
- **关闭**
  - 申购单关闭操作
- **详情页**
  - 申购单基本信息
  - 明细行列表（含采购状态、回料状态）
  - 关联采购单列表

### 3.3 采购申请
- **列表页**
  - 搜索筛选（按采购单号、状态、供应商）
  - 分页展示
  - 操作：处理/关闭/详情
- **采购处理页**（采购员操作）
  - 填写采购数量
  - 填写预计到货时间
  - 填写单价
  - 提交采购单
- **关闭**
  - 采购单取消/关闭
- **详情页**
  - 采购单基本信息
  - 采购明细列表
  - 关联回料情况

### 3.4 采购管理
- **列表页**
  - 搜索筛选（按采购单号、状态）
  - 分页展示
- **详情页**
  - 采购单完整信息
  - 到货跟踪

### 3.5 回料管理（不做质检）
- **列表页**
  - 搜索筛选（按回料单号、状态）
  - 分页展示
- **新增回料**
  - 选择回料物料（从采购明细选择）
  - 只选择仓库（不做库位）
  - 填写实收数量
  - 自动生成入库单
- **入库确认**
  - 确认实收数量
  - 生成新批次
  - 更新库存
- **详情页**
  - 回料单信息
  - 入库记录

## 四、生产

### 4.1 生产列表（返工生产单列表）
- **列表页**
  - 搜索筛选（按生产单号、返工单号、状态）
  - 分页展示
  - 状态：草稿/待领料/生产中/已完成/已关闭
  - 操作：发料列表/配制/详情/强制关闭
- **强制关闭**
  - 任意状态可强制关闭
  - 填写关闭原因
  - 单据置为"已关闭"

### 4.2 发料列表（领料申请 → 发料）

#### 4.2.1 领料申请（生产员发起）
- **列表页**
  - 按返工生产单展示领料申请
  - 状态：草稿/审批中/已审批/已驳回/部分发料/已发完/已关闭
- **新增领料申请**
  - 选择返工生产单（自动带出BOM物料）
  - 设置批次（无需设备）
  - 领返工产品批次
  - 填写申领数量（参照BOM配比）
  - 实时显示仓库可用库存
  - 提交审批
- **领料审批**
  - 审批人查看领料申请
  - 审批通过/驳回
- **退料申请**（领料后多领退回）
  - 发起退料申请
  - 填写退料原因
  - 填写申请退料数量
  - 关联原领料批次（可选）
  - 提交审批

#### 4.2.2 发料执行（仓库发料）
- **列表页**
  - 待发料列表（已审批领料申请）
  - 已发料历史
- **发料操作**
  - 选择领料申请
  - 指定发料批次（从库存批次选择）
  - 填写实发数量
  - 扣减库存
  - 完成发料
- **退料入库**（仓库确认退料）
  - 选择退料申请
  - 确认实退数量
  - 入库（可能沿用原批次或新建批次）
  - 增加库存

### 4.3 配制（返工生产执行）

#### 4.3.1 步骤备料
- **步骤列表**
  - 展示当前生产单的工艺步骤
  - 状态：待备料/备料中/已完成
- **备料操作**
  - 选择步骤
  - 查看BOM步骤投料明细（物料、计划备料数量）
  - 实际称量记录（实际称量数量）
  - 选择物料批次
  - 完成备料
- **备料详情**
  - 备料记录查看

#### 4.3.2 步骤投料
- **步骤列表**
  - 展示已备料步骤
  - 状态：待投料/已投料
- **投料操作**
  - 选择步骤
  - 选择投料模式：
    - 模式1-直接投料：录入物料批次和数量
    - 模式2-备料后投料：引用备料子单，自动带出批次数量
  - 填写实际投料数量（允许与备料有差异）
  - 完成投料
- **投料详情**
  - 投料记录查看

### 4.4 生产入库
- **入库发起**（生产员手动发起）
  - 选择返工生产单
  - 填写入库数量
  - 选择入库仓库
  - 系统自动生成批次（返工批次规则）
- **入库执行**（仓库确认）
  - 查看入库单
  - 确认实入数量
  - 更新库存
  - 完成入库
- **入库详情**
  - 入库记录查看

## 附录：状态流转

### 返工单
```
草稿 → 审批中 → 已审批（生产单和申购单变为可执行）
            ↑
      驳回 → 退回草稿
```

### 返工生产单
```
草稿 → 待领料 → 生产中 → 已完成
                     ↑
               强制关闭 → 已关闭（任意阶段可操作）
```

### 申购单
```
草稿 → 待确认 → 已确认 → 已关闭
```

### 领料申请
```
草稿 → 审批中 → 已审批 → 部分发料/已发完 → 已关闭
            ↑
      驳回 → 退回草稿
```
